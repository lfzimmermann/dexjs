# dexjs - self-hostable Pokedex for Pokemon Yellow Edition

Just a quick and (very, very) dirty webapp, because I wanted something to track my process on.

Caches requests to [pokeAPI](https://pokeapi.co) to conform to their rules and regulations.

Working example can be found on my website https://kagenao.de/pokedex.

### Running

To run it simply type ```npm install``` and then ```npm run start```.

It will be reachable via ```http://localhost:25666```
